# 智能合约开发
## contract 合约代码
## dapp 基于web3js 实现对上contract 合约的查询和调用
## 进入dapp 进行npm install

主链：大家熟悉的BTC ETH 等，用不同技术实现的底层区块链

区块链浏览器：给用户直观的查看主链上发生的数据

钱包：存储币，和用于DApp 支付

GAS：矿工费，发起区块链转账和合约调用支付的手续费
web3js：一个js 框架用于和dapp 交互
web3j：一个java 框架用于和dapp 交互
Remix：部署合约的工具
solidity：ETH 合约开发语言
